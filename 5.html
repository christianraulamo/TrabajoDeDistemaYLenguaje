Instalación de Drivers de adaptadores WiFi.

Cuando recibe en casa un nuevo adaptador WiFi USB necesitará instalar el driver apropiado para el sistema operativo. Puedes hacerlo desde el CD de la caja, pero si quieres actualizar la última versión lo mejor es ir a, el sitio web del fabricante del adaptador y descargar el controlador para ese modelo y versión exactos.

Todos los adaptadores por USB de WiFi son un dispositivo periférico, y como tal requiere un controlador de software, o “driver” para funcionar. Por eso, cuando conecta el adaptador a un puerto USB, no siempre se inicia automáticamente. Afortunadamente, la mayoría de los adaptadores WiFi llevan en la caja un CD con el software necesario.

Sin embargo, dado que un número creciente de ordenadores portátiles y de sobremesa no tienen unidades de lectura de CD, lo mejor es comprobar siempre las versiones más recientes en el sitio web del fabricante del chip del adaptador WiFi.

** Consejo para instalar los drivers de adaptadores WiFi en Windows 8 y Windows 10

Si acabas de instalar los controladores para un adaptador inalámbrico y no parece funcionar, lo mejor es desinstalarlos, reiniciar el equipo y hacer la instalación desde el principio. Esta vez sin embargo, preste atención a la funcionalidad que tienen los sistemas operativos Windows 8 y Windows 10 para instalar software con compatibilidad para sistemas anteriores.

Por ejemplo, si nuestro chip WiFi como el RTL3070, tiene un driver que solo está actualizado hasta Windows 7, significa que puede funcionar perfectamente en Windows 10 si usamos la compatibilidad del driver para versiones anteriores de Windows en Windows 10. Si tenemos el driver localizado en una carpeta de nuestro disco duro o en el CD, con el botón derecho del ratón hacemos clic y vemos las propiedades del software, luego seleccionamos instalar en modo compatibilidad para Windows 7 y listo. Cuando termine la instalación funcionará el chip WiFi perfectamente en Windows 10. Todos los detalles en esta guía de instalación de drivers para Windows 10.
